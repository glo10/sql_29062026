Affichez les catégories et les employés depuis la base de données dans EXCEL.

1. Il faut installer les connecteurs Excel dédié à chaque base de données directement proposées par Excel ou sinon il faut faire une petite recherche sur Google en tapant par exemple "connecteur Excel PostgreSQL"
2. Suivez les étapes des articles ci-dessous selon votre cas d'usage
- [Vidéo import via le connecteur MySQL](https://www.youtube.com/watch?v=nIouAGiDbvY)
