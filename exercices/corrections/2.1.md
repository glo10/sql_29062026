# Exercice 2.1 : export des données en CSV

1. Exportez les fournisseurs dans un ficher .CSV

```bash
# Postgres depuis un client Shell par exemple PowerShell
psql -U postgres -d northwind -c "\copy fournisseurs TO 'fournisseurs.csv' WITH CSV HEADER"
# La console va vous demander le mot de passe ensuite si c'est correct, l'export est effectué
```
- Possible d'effectuer un export depuis l'interface graphique PgAdmin4
    - ***Databases > Northwind > Schéma > Public > Tables > fournisseurs > Clic droit > Import/Export Data > Export > OK***

```bash
# MySQL/MariaDB depuis un éditeur SQL
SELECT * FROM fournisseurs
INTO OUTFILE "c:/export.csv"
FIELDS TERMINATED BY ','
ENCLOSED BY '"'
LINES TERMINATED BY '\n';
```

```bash
# MYSQL/Maria DB depuis une console (powershell par exemple ou autre client Shell)
mysql -u root -p -e "SELECT * FROM fournisseurs;" northwind > fournisseurs.txt
# La console va vous demander le mot de passe ensuite si c'est correct, l'export est effectué
```
- Possible d'effectuer un export depuis l'interface graphique dédié par exemple Heidi SQL, PhpMyAdmin, Adminer, etc.
```bash
# SQL SERVER
sqlcmd -S localhost -d northwind -U root -P root -Q "SELECT * FROM clients" -s "," -o "clients.csv"
```
