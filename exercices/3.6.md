# Exercice 3.6 : les jointures avec JOIN

1. Affichez la liste des produits avec le nom de leur catégorie respective.
2. Affichez la liste de tous les clients, avec les IDs des commandes qu'ils ont passées. Triez le résultat pour afficher en premier les clients qui n’ont jamais commandés.
3. Affichez les noms , prénoms des employés et l'ID des commandes qu'ils ont gérées.
On veut s'assurer d'afficher toutes les commandes, même si (théoriquement) une commande est forcement rattaché à un employé.
4. Affichez toutes les combinaisons possibles entre les produits et les catégories.
5. Affichez l'ID de la commande, le nom du client qui a acheté, le nom de l'employé qui a fait la vente, et le nom du produit acheté.
